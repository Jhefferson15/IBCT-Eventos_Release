# Histórico de Versões

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [v1.0.5+6] - 30/12/2025

### Novidades
- **Nova Página de Perfil**: Agora você pode gerenciar suas informações de conta através de uma interface dedicada e completa.
- **Sistema de Atualização Inteligente**: O aplicativo agora exibe o progresso das atualizações em tempo real com uma nova barra de progresso visual.
- **Opção de Download Manual**: Caso a atualização automática falhe, o sistema oferece automaticamente uma opção para baixar a nova versão diretamente pelo navegador.
- **Melhorias na Loja e Participantes**: A edição de itens e o gerenciamento de participantes foram aprimorados para serem mais ágeis e intuitivos.

### Melhorias
- Correções visuais e de fluxo para garantir uma experiência de uso mais fluida durante a edição de eventos.
- Ajustes internos para aumentar a estabilidade geral do aplicativo.

## [v1.0.3+4] - 29/12/2025

### Correções
- Ajustes na estabilidade do processo de distribuição de novas versões.
- Melhorias na confiabilidade das requisições de rede.

## [v1.0.2+3] - 29/12/2025

### Lançamento do Portal Web
- **Novo Painel Administrativo**: Uma interface web completa para gestão de eventos, visualização de vendas e métricas.
- **Análise de Eventos**: Gráficos e relatórios detalhados sobre o desempenho dos seus eventos.
- **Editor de Eventos Web**: Crie e edite seus eventos diretamente pelo computador com mais conforto.

### Novas Funcionalidades
- **Importação em Massa**: Adicione múltiplos participantes de uma só vez utilizando o novo modelo de planilha CSV (`modelo_importacao.csv`).
- **Nova Tela de Login**: Experiência de entrada renovada, moderna e mais segura.
- **Compatibilidade Web**: O sistema agora suporta instalação como aplicativo web (PWA) no seu computador.

## [v1.0.0+1] - 12/10/2025

### Lançamento Inicial
- Estrutura base do aplicativo.
- Funcionalidades essenciais para gerenciamento de eventos.
